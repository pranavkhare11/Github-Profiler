# GitHub Profiler

A modern web application that helps you analyze and visualize your GitHub profile statistics and contributions.

## 🎯 Overview

GitHub Profiler is a React-based tool designed to provide insightful analytics about your GitHub account. It fetches data from the GitHub API and presents comprehensive statistics in an intuitive, visually appealing interface.

## 🚀 Features

- **Profile Analytics**: View detailed statistics about your GitHub profile
- **Contribution Insights**: Analyze your contribution patterns and trends
- **Repository Overview**: Get insights into your repositories
- **Interactive Dashboard**: Clean and modern UI built with React
- **Real-time Data**: Fetch live data from GitHub API

## 🛠️ Tech Stack

- **Frontend**: React 19.2.7
- **Build Tool**: Vite
- **Language**: TypeScript (52.5%)
- **Styling**: CSS (45.9%)
- **HTTP Client**: Axios
- **Routing**: React Router
- **Linting**: ESLint & Oxlint
- **Dev Tools**: Babel, TypeScript Compiler

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Setup

```bash
# Clone the repository
git clone https://github.com/pranavkhare11/Github-Profiler.git
cd Github-Profiler

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📜 Available Scripts

- `npm run dev` - Start the development server with hot module replacement
- `npm run build` - Compile TypeScript and build for production
- `npm run lint` - Run ESLint and Oxlint for code quality checks
- `npm run lint:fix` - Automatically fix linting issues
- `npm run preview` - Preview the production build locally
